# 🛍️ React E-Commerce App

A basic **E-commerce web application** built using **React.js** and **Vite**.  
This project was developed during a **45-day MERN stack training program** but currently includes only the **frontend** portion (no backend integration).

---

## 🚀 Features

- 🛒 Product Listing Page  
- ➕ Add to Cart  
- ➖ Remove from Cart  
- 💰 Display Total Price and Quantity  
- 🔀 Page Navigation with React Router  
- ⚛️ Component-Based Architecture  
- 🧠 State Management with `useState` and `useContext`  
- ⚡ Fast Build and Hot Reload using **Vite**  
- 📱 Basic Responsive Design

---

## 🧑‍💻 Tech Stack

| Technology     | Description                      |
|----------------|----------------------------------|
| React.js       | Frontend Library                 |
| Vite           | Fast Development Build Tool      |
| React Router   | Client-side Routing              |
| CSS            | Styling (custom or basic CSS)    |
| LocalStorage   | Frontend Data Persistence        |
